## CS4375-Decision-Tree-Random-Forest-Classification-Model

A machine learning project implementing and comparing tree-based classifiers on two datasets: the Pima Indians Diabetes dataset (binary classification) and an Asian Cuisines dataset (multi-class classification).

### Models Explored:
- Decision Tree
- Bagging
- Random Forest
- AdaBoost

### Key Results: 
Diabetes dataset: Bagging performed best (~79%) vs. single Decision Tree (~68%)

Cuisines dataset: Tuned Random Forest achieved ~83% accuracy vs. single Decision Tree (~64%)

### Tools:
Python, scikit-learn, pandas, matplotlib, graphviz
